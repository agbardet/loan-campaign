# AllLife Bank — Personal Loan Campaign

**Predicting which liability customers will accept a personal loan offer — achieving 90.3% recall with a post-pruned Decision Tree.**

---

## Key Results

| Metric | Value |
|--------|-------|
| Test F1 | **0.915** |
| Test Recall | **0.903** |
| Test Precision | **0.929** |
| Overfitting gap (Train–Test F1) | 3.1 pp |

**Top 3 features:** Income · Family size · Education level

---

## Business Context

AllLife Bank wants to grow its loan portfolio by cross-selling personal loans to existing liability (deposit) customers. With only 9.6% of the 5,000-customer dataset having accepted a previous offer, mass outreach is expensive and low-conversion. A propensity model lets the marketing team target the right customers at the right time — reducing cost per acquisition and increasing campaign ROI.

## Running the Notebook

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Install src/ as a local package (so notebook imports work)
pip install -e .

# 3. Run and export the notebook
jupyter nbconvert --to html --execute notebooks/loan_campaign.ipynb \
  --output-dir docs --output loan_campaign
```

Or open `notebooks/loan_campaign.ipynb` in Jupyter and run all cells.

## Dataset

`data/Loan_Modelling.csv` — 5,000 AllLife Bank customers, 14 columns.  
See [data_dictionary.md](data_dictionary.md) for full column descriptions.

**Target:** `Personal_Loan` (0 = declined, 1 = accepted) — 9.6% positive class.
